# RMECloud
Connect RME around the world

## Local run

### Requirement

*    Erlang
*    Yaws
*    Make (for Linux users)
*    MingW32 (for Windows users)

### Clone and launch

**For Linux**
```
git clone https://github.com/RMEx/RMECloud
cd RMECloud/webapp
mkdir log beam
make
```

**For Windows**
```
git clone https://github.com/RMEx/RMECloud
cd RMECloud/webapp
mkdir log beam
mingw32-make
```
